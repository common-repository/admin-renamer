== Description ==

Wtyczka pozwala zmienić login domyślnego użytkownika "admin".

== Installation ==

1. Skopiuj wszystkie pliki do katalogu **/wp-content/plugins/admin-renamer/**
2. Aktywuj plugin w panelu administracyjnym
3. Zmień login konta **"admin"** w menu **Ustawienia/Admin Renamer**
4. Ze względów bezpieczeństwa, po zmianie loginu wyłącz i usuń wtyczkę!
